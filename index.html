Audio Troubleshoot

Check if headset is Poly Blackwire

If Poly Blackwire request new Headset - use staples requitsition not Alight IT

Request only Jabra or Logitech

If not Poly Blackwire check audio setting in Five9

Check if correct audio device is selected

If settings are good in Five9 check Windows sound settings via Control Panel

Check audio levels and mute status of headphones and mic

If headphones work but not mic run online mic test

If mic fails online mic test try restarting the laptop and try test again before requesting new headset

If online mic test passes then this is an issue isolated to Five9. Have user log out and back into Five9

If settings are good in Windows try unplugging and plugging back in the headset
